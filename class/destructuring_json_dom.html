📦 Destructuring in JavaScript

What is Destructuring?

Destructuring is a convenient way to extract values from arrays or objects and assign them to variables. It makes code cleaner and more readable.

1️⃣ Array Destructuring

// Basic Array Destructuring const numbers = [10, 20, 30]; const [a, b, c] = numbers; console.log(a, b, c); // Output: 10, 20, 30 // Skip Elements const [first, , third] = [1, 2, 3]; console.log(first, third); // Output: 1, 3 // Rest Operator const [x, ...rest] = [1, 2, 3, 4]; console.log(rest); // Output: [2, 3, 4]

2️⃣ Object Destructuring

// Basic Object Destructuring const person = {name: "Alice", age: 28}; const {name, age} = person; console.log(name, age); // Output: "Alice", 28 // Rename Properties const {name: n, age: a} = person; // Default Values const {city = "NYC"} = person; // Nested Objects const {address: {city}} = person;
Click button to see output ↓

3️⃣ Shallow Copy vs Deep Copy

// SHALLOW COPY const original = {x: 1, nested: {y: 2}}; const shallow = {...original}; shallow.nested.y = 999; console.log(original.nested.y); // 999! // DEEP COPY const deep = JSON.parse( JSON.stringify(original) ); deep.nested.y = 999; console.log(original.nested.y); // Still 2

📄 JSON Handling

What is JSON?

JSON (JavaScript Object Notation) is a lightweight data format. It uses key-value pairs and is language-independent, making it perfect for APIs and data storage.

Data Type Example Description
Object {"name": "John"} Key-value pairs
Array [1, 2, 3] Ordered list
String "Hello" Text in quotes
Number 42, 3.14 Integer or decimal
Boolean true, false True or false
Null null No value

🔄 JSON.stringify() - Object to String

🌳 DOM Manipulation

What is the DOM?

DOM (Document Object Model) is a tree structure representing your HTML page. JavaScript can access and modify every element in this tree.

1️⃣ DOM Tree Structure

document
├── html
│   └── body
│       ├── header
│       ├── main
│       │   ├── h1 (Text: "Title")
│       │   ├── p (Text: "Content")
│       │   └── button (class="btn")
│       └── footer

2️⃣ DOM Selectors

Method Example Returns
getElementById() document.getElementById("main") Single element
querySelector() document.querySelector(".btn") First match
querySelectorAll() document.querySelectorAll(".item") All matches
getElementsByClassName() document.getElementsByClassName("active") HTMLCollection
getElementsByTagName() document.getElementsByTagName("p") HTMLCollection

3️⃣ Read/Write HTML & CSS

Property/Method Purpose Example
innerHTML Get/Set HTML content el.innerHTML = "<p>New</p>"
textContent Get/Set text only el.textContent = "Hello"
style Get/Set inline CSS el.style.color = "blue"
classList Manage CSS classes el.classList.add("active")
getAttribute() Get attribute value el.getAttribute("id")
setAttribute() Set attribute el.setAttribute("id", "main")

4️⃣ Creating & Appending Elements

Steps:

  1. Create: const el = document.createElement("tag")
  2. Configure: el.textContent = "text"; el.id = "id"
  3. Append: parent.appendChild(el)

5️⃣ Deleting Elements

Method Syntax Use Case
remove() element.remove() Remove element directly
removeChild() parent.removeChild(child) Remove from parent
innerHTML = "" element.innerHTML = "" Clear all content
replaceChild() parent.replaceChild(new, old) Replace one element
